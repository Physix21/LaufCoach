# Garmin Rohdaten

Hier Garmin-Exporte ablegen.

Mögliche Formate:

- .fit
- .tcx
- .gpx
- .csv

Empfehlung:

- Einzelne Aktivitäten als .fit oder .tcx exportieren, falls verfügbar.
- Alternativ Wochen-/Aktivitätsübersichten als CSV ablegen.
- Dateinamen möglichst nach Datum und Sportart:

```text
2026-07-08_run_8x400.fit
2026-07-10_bike_5x4min.fit
2026-07-12_run_easy.fit
```
